Files in this folder



This directory contains the paper:

    Kuhner, M. K. and J. Felsenstein.  1994.  A simulation comparison of
       phylogeny algorithms under equal and unequal evolutionary rates.
       Molecular Biology and Evolution  11: 459-468.

The files are:

 simulation.tex         LaTeX input file for the manuscript
 simulation.dvi         LaTeX DVI device-independent output file
 simulation.ps          Extended Postscript version of manuscript

 simulation.pdf         PDF version of manuscript
